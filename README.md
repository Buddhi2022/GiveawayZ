# Self-Host üõ†

- Prequisities

```sh

pkg install wget -y && pkg install python -y

```

- Install PIP Requirements

```sh

pip install -U requirments.txt

```

- Copy and set variables in ```env.sample``` and save as ```local.env```.

- Run the bot

```sh

python3 bot.py

```
